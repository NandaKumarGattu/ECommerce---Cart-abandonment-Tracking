# ECommerce - Cart-abandonment-Tracking

ECommerce website integrated with cart abandonment tracking with API documentation, Swagger UI, Email notification tracking, Database Integration. It is a full fledged frontend and backend project. 
The main programming languages and frameworks are used 
Springboot - A Java Framework
  - Spring Security
  - RESTful APIs and Microservices
  - Dependency Injection and IoC
Express.js - Web Application framework
HTML, CSS, JS - For Frontend
MongoDB - storing data of user, products, data of cart abandonments
